KreatorJS

Uma ferramenta criada para ceder a necessidade da criação de algo que desenvolve um website e ainda depois o exporta no modelo aplicação desktop. A ideia desse projeto foi meio inesperada porque o principal motivo por essa ferramenta existir foi pela ideia de usar 100% a capacidade de IA'S para o desenvolvimento de aplicações.

KreatorJS é feita usando HTML, JavaScript, CSS e NodeJS por meio da tecnologia Electron.

  • Detalhes 31/08/2025
    A ferramenta permite adicionar componentes como botões, textos, imagens e etc. Também é possível colocar ações nos componentes como: Quando forem clicados, mudado o estado deles e muito mais. Modificar o estilo dos componetes com características do CSS e mudar eles também por eventos.
